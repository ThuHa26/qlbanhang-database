# Sales Management Database

## Description
This project is a sales management database system built using SQL Server.  
It manages customers, products, invoices, and order details.

## Features
- Designed relational database with multiple tables
- Applied PRIMARY KEY and FOREIGN KEY constraints
- Wrote SQL queries using JOIN, GROUP BY, HAVING, subqueries
- Performed data analysis (revenue, customer ranking)

## Technologies
- SQL Server
- SQL

## Files
- data.sql: database schema and sample data
- queries.sql: SQL queries for analysis
